# MS SQL Server + Adminer deployment

## Env
Put env vars to `.env`.

Example in `.env.example`.

## Run
```bash
make run
```

## Stop
```bash
make stop
```

## Stop and Remove containers
```bash
make down
```

## Clean
```bash
make clean
```
- to clean volumes (requires sudo)
